# Simulation for Memory Management

## Usage

Configure the follow values as you wish in `simlru.h`.

```
// simulation control
MEM_SIZE
PROC_MAX
PROC_SIZE_MAX
PAGE_SIZE 
PAGE_NUM
RSS 10

// output control
PRINT_PAGE
PRINT_LRU
```

Then build the source and start your test.

```
make
./test
```
